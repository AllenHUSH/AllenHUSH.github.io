# 我只是想要一台办公机——1500元的AMD YES！

# 前言

这次的客户很特别，是我爸。

先简单介绍一下我爸。男，汉族，财务工作。一直用的办公古董机突然罢工了，在我不断蛊惑之下终于来到我这里装机了。年轻的时候也是个搞机玩家，那时候AMD是弟弟的代名词，Intel一家独大，所以在选择配置的时候，我们有以下对话：

-1400？AMD的？

-AMD的，AMD YES啊。

-(看着我选的塔式散热器)怪不得说AMD热量大。

-爹，这是能压制135W功耗的，你这个U TDP才65…

那么问题来了，我为什么选择一个这么强力的散热器来压制一颗1400呢，因为：

  1. 办公室用机指定机箱，机箱内部空间充足，还用不上光污染；

  2. 预算相对于性能要求来说过于富裕，多花几十块钱放在散热上未尝不可；

  3. 我爸习惯7*24小时不关机，积热当然越少越好；

以及等等角度考虑，最终配置敲定如下——

# 配置

这个仅要求基本办公的主机配置单中，Ryzen1400和8代i5、9代i5基本持平，4核8线程又是比Intel同级别CPU的优势，处理解压缩等工作时有明显的优势；

GPU仅仅是预算充足且求稳，选择了讯景的全新库存卡，对于游戏玩家来说RX550根本瞧不上眼，对于办公用户来说又显得富裕；对比某宝一众品牌整机，¥4000价位的主机仅仅配置GT710，良心不会痛嘛；

内存方面，选择了七彩虹的内存；便宜的内存条中它是最有背景的，有背景的内存条中它是最便宜的；当然，再加二三十元，完全可以上马甲条、灯条，可以，但没必要；

主板是随便搜索a320m后挑了一个最便宜最靠谱的，没想到到手之后发现，影驰这款入门级主板既带主板呼吸灯又带NVMe接口，虽然不是什么新鲜东西，但200块钱的主板要啥自行车；

固态选择了全网爆款影驰战将，120G容量装系统盘特别富裕，原本¥119的价格放在购物车几天降价为¥99，还敢再便宜点么？

其他硬件没什么好说的，仅仅是电源，实际上爱国者的G3电源完全够用且靠谱，但总感觉电源不过百就会红星闪闪…总而言之，一套配置下来，这已经是一台能低画质吃鸡、高帧数打LOL的游戏办公机了。

接下来，让我们来装个机吧！
