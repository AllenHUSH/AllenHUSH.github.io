# Node.js

TIP

万物皆可js.

第一次写node的时候，一切似乎很完美——返回一个表单、填写表单、提交数据、插入云数据库。但现在它变了。呵，node.

先是告诉我query不是一个属性，connection未定义，接着控制台建议我升级mysql版本。有没有搞错，我是昨天才安装的最新版本。

其实这是mysql8.0的锅，出于安全策略mysql会对密码进行加密，所以即便我连接密码是正确的，也会给我报错。

？？？

Ok，修改安全策略，修改密码安全程度，给我所有处心积虑的密码打分为50分——也就是不安全。当我去请教一位后端程序员，“这玩意儿可咋整啊”，他回答我：“我的密码就是mysql。”

？？？

Oracle费尽心思让用户的数据库变得安全，程序员用户又表示我-就-不。

Ok，一番折腾。Connection不再为定义了，控制台又告诉我跨域。

再见👋我改bug去了。
