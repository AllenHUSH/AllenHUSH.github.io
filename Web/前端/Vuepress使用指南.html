# Vuepress使用指南

Allen

Vuepress官方网站

# 介绍

正如官方介绍，Vuepress是Vue 驱动的静态网站生成器；Markdown转HTML不是一件稀罕事，Vuepress只是提供了一种途径，外带约定（约定结构、主题等）和可以使用Vue组件的便捷。

# 安装及使用

# 安装
yarn global add vuepress # 或者：npm install -g vuepress

# 新建一个 markdown 文件
echo '# Hello VuePress!' > README.md

# 开始写作
vuepress dev .

# 构建静态文件
vuepress build .

# 自动构建目录

npm install -D boboidream/vuepress-bar

// .vuepress/config.js
module.exports = {
    ...
    themeConfig: {
        sidebar: 'auto',
        sidebarDepth: 2,
        serviceWorker: true
    },
    plugins: ['permalink-pinyin', ['autobar', { 'pinyinNav': true }]],
    ...
}

docs文件夹下，对所有文件夹进行目录分类，nav开头的文件夹进如入导航栏，其他侧边栏，其他文件默认分类。
