# 前端

前端其实挺有趣的。

我相信大多数计算机专业都不会认真教web，特别是web前端。我们学校也仅仅是开设了一学期的web课程，但我并没有听。

HTML和CSS是极简单的语言，HTML注重结构，CSS注重层叠。听说隔壁班还学了PHP，你看，不难发现，老师们在教“做网页”，也不是在教“做web”。Web这个词可大着呢。

学完HTML和CSS之后，其实挺迷茫的。那段时间在努力尝试写出高端大气的网页来，企图来摆脱“切图仔”这个名声。再后来接触了Vue，接触了node，觉得自己总算离这个名字越来越远。

再再后来，不去想这个问题了，因为前端的世界太有趣了，我可以手指头在键盘上飞舞一整天，了解到新的技术和标准就开心到不行。

再再再后来，也就是现在，我觉得，前端有趣的地方在于——这份职业本身就是切图仔。代码与设计，两种美学的碰撞，0和1的世界里也可以丰富多彩。

现在是2019年12月7日，给自己定一个小目标——截止到毕业时

框架：Vue React Express KOA Egg

语言：TypeScript Less Sass Stylus

技术：PWA Electron

项目：1套VueUI库 1个完整优雅的全栈项目 1个无用的开源项目
