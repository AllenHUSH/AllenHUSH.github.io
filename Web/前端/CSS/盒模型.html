# 盒模型

参考文档-菜鸟教程

参考文档-MDN

盒模型是每一个前端入门都听过的问题，我们先来简单回顾一下：

不同部分的说明：

  • Margin(外边距) - 清除边框外的区域，外边距是透明的。
  • Border(边框) - 围绕在内边距和内容外的边框。
  • Padding(内边距) - 清除内容周围的区域，内边距是透明的。
  • Content(内容) - 盒子的内容，显示文本和图像。

以上盒模型我们称之为W3C盒模型，也是目前被普遍接收的。在IE低版本中，还有怪异的IE盒模型。

# 浏览器的兼容问题

# IE5/6宽度

IE5.X 和 6 在怪异模式中使用自己的非标准模型。这些浏览器的 width 属性不是内容的宽度，而是内容、内边距和边框的宽度的总和。

虽然有方法解决这个问题。但是目前最好的解决方案是回避这个问题。也就是，不要给元素添加具有指定宽度的内边距，而是尝试将内边距或外边距添加到元素的父元素和子元素。

# IE8之前

IE8 及更早IE版本不支持设置内边距的宽度和边框的宽度属性。

解决IE8及更早版本不兼容问题可以在HTML页面声明 <!DOCTYPE html>即可。

# box-sizing

box-sizing是CSS3新增的属性，目前被广大浏览器接纳的有两个合法值：

  • content-box即默认值，同W3C盒模型；
  • border-box同IE盒模型。
