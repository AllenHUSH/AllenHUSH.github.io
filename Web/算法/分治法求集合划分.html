# 分治法求集合划分

# 问题描述

F(n,m)表示把n个元素的集合分为m个子集，有多少种分法？

# 重点

理解一下return count(ln - 1, lm - 1) + lm * count(ln - 1, lm)

n个元素的集合可以划为F（n,m）个不同的由m个非空子集组成的集合。

考虑3个元素的集合，可划分为：

  1. 1个子集的集合：{ {1，2，3} }

  2. 2个子集的集合：{{1，2} ，{3}} ， {{1，3}，{2}} ， {{2，3}，{1}}

  3. 3个子集的集合：{{1}，{2}，{3}}

所以 F（3，1）=1 　　 F（3，2）=3 　　 F（3，3）=1

如果要求F（4，2）该怎么办呢？

A.往①里添加一个元素 {4} ，得到{{1，2，3}，{4}}

B.往②里的任意一个子集添一个4，得到

{{1，2，4}，{3}} ， {{1，2}，{3，4}}

{{1，3，4}，{2}} ， {{1，3}，{2，4}}

{{2，3，4}，{1}} ， {{2，3}，{1，4}}

所以F（4，2） = F（3，1）+2*F（3，2） = 7

以此推广得，F（n，m） = F （n-1，m-1）+ m * F（n-1，m）

参考文档 代码实现

/**
 * 分治法
 * n个元素m个非空子集
 * 可以组成多少种集合
 * 即F(n,m)表示把n个元素的集合分为m个子集
 * 有多少种分法
 * 
 * @author AllenHUSH
 */

/**
 * 声明全局变量
 * 开始
 */
const n = 4       // 全集包含n个元素
const m = 2       // m个非空子集

/**
 * 计算函数
 * 开始
 */
let count = (ln, lm) ={
        if (lm == 1) {
            //即每个集合仅包含1个元素，n个元素分成n个集合，共1种组合方式
        return 1
    } else {
            if (ln == lm) {
                //即每个集合包含n个元素，n个元素分成1个集合，共1种组合方式
            return 1
        } else {
                //核心语句
            return count(ln - 1, lm - 1) + lm * count(ln - 1, lm)
        }
    }
}

/**
 * 主函数入口
 * 开始`
 */
let main = () ={
        let ans = count(n, m)
    console.log("存在 " + ans + " 种集合方式");
}
/**
 * 程序入口
 * 执行main函数
 */
main()
